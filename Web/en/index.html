The Baka Keyboard Project

Baka Keyboard is a mechanical keyboard with only two keys.

This is maybe both a useless project and a useful project.

参与众筹

Technical Details

We provide Cherry Blue (Recommended) / Red / Black / Green / White Switches for your Baka Keyboard.

For main controller, ATmega328P and ATtiny13 are available.

3D printed PLA Black and PLA White case. For keycap ABS White and PBT Red are provided. (Recommended: Black and White / White and Red)

Baka Keyboard is all-key programmable.

Design

Well-designed case allows great user experience even on normal surfaces.

Usage

Baka Keyboard can be used as OSU! supplement, iwanana supplement, 0/1 keyboard (used for coding?), or for quickly waking up or sleeping your OS, quickly adjusting screen brightness, or used with Photoshop / After Effects and etc.

Programming port is provided and controlling software is open source, updated and maintained. This is how we make useless things useful.
